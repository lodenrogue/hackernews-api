# hackernews-api


### Services available

```
/hackernews/top
/hackernews/ask
/hackernews/best
/hackernews/jobs
/hackernews/new
/hackernews/show
```

Limit results with optional query parameter

```
/hackernews/top?limit=5
```

### How to build
```
./gradlew build
```

### How to run
```
java -jar build/libs/hackernews-api-<version>.jar
```
